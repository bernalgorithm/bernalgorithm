# Project Goals & Persona

This project is a personal portfolio and blog site built with Next.js and Tailwind CSS. 

## Primary Goals
1. **Resume Extension**: Serve as a high-quality, professional landing page for potential employers, highlighting my experience as an iot/embedded/firmware engineer who also contains modern web development chops.
2. **Project Documentation**: Provide a "lab notebook" for my development projects, focusing on technical depth and architectural decision-making.
3. **Skill Demonstration**: Act as a showcase of modern web development best practices (e.g., performance, accessibility, static generation).

## Technical Constraints
- **Architecture**: Next.js using static site export for deployment on GitHub Pages.
- **Content**: Markdown-based blogging with `gray-matter` and `next-mdx-remote`.
- **Environment**: Node.js v20.12.0 (managed via `.nvmrc`).
- **Styling**: Tailwind CSS.

## Tone & Style
- Professional, technically rigorous, and clear.
- Emphasize "the why" behind technical choices, especially when bridging firmware concepts with web implementations.
